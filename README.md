# 📚 Моя библиотека

Персональный менеджер книг на **Vue 3** + **Bootstrap 5**.

## Функционал

- Добавление книг с названием, автором, жанром, описанием и обложкой
- Загрузка обложки — при добавлении или кликом на карточку
- Оценка от 1 до 5 звёзд
- Отметка книги как прочитанной
- Удаление книг из коллекции
- Поиск по названию и автору
- Фильтрация: Все / Прочитано / К прочтению / С оценкой
- Сортировка: по дате, названию, автору, оценке
- Статистика: всего книг, прочитано, в очереди, средняя оценка
